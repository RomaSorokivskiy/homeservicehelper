# Апаратна платформа

## Цільова збірка без жорсткого бюджетного обмеження

Це базовий варіант для AMP, Jellyfin, домашньої панелі, Home Assistant і побутових сервісів. Він більший за NUC, зате не потребує ненадійної USB-шафи для основного сховища.

Для фактичного бюджету до 35 000 грн використовується перевірена refurbished workstation-платформа. Конкретний актуальний кошик наведений у [purchase-list-35000.md](purchase-list-35000.md); конфігурація нижче залишається орієнтиром для майбутнього повного апгрейду.

## Чому не готовий NUC як основний варіант

NUC/mini-PC підходить для обчислень, але переносить головну проблему у зовнішню USB-шафу: більше блоків живлення, кабелів, точок відмови та складніше коректне відновлення дисків. Його варто брати лише якщо медіатека житиме на окремому готовому NAS із власним Ethernet.

## Альтернатива з мінімальним енергоспоживанням

Mini-PC на Intel Core Ultra 5 U/H із 64 GB RAM + окремий 4-дисковий NAS. Це тихіше й компактніше біля робочого місця, але дорожче як система та має дві окремі точки адміністрування. Intel документує Quick Sync і підтримку до 128 GB для частини мобільних Core Ultra Series 2; конкретний mini-PC треба перевіряти за лімітом виробника.

## Розподіл ресурсів

- 2 CPU-потоки та 4 GB RAM зарезервувати для ОС, VPN, DNS, proxy і моніторингу.
- Home Assistant та критичні побутові сервіси: гарантовані 4–6 GB RAM, без агресивного overcommit.
- Jellyfin: апаратний доступ до Intel iGPU; тимчасовий каталог транскодування на NVMe.
- AMP: решта CPU/RAM із лімітами на кожну інстанцію; новий ігровий сервер не запускається, якщо порушить критичний резерв.
- Vaultwarden, Mealie, Vikunja та Homebox: окремі БД/volume і окремі резервні копії.

## Що уточнити перед покупкою

1. Максимальний одночасний набір ігор та кількість гравців.
2. Початковий і бажаний через три роки обсяг медіатеки.
3. Місце встановлення та допустимий шум уночі.
4. Бюджет без HDD і окремо бюджет на диски/UPS.
5. Точна сумісність кулера, SFX-БЖ, HBA і кабелів із ревізією корпусу.
